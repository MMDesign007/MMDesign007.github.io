Ruby BASICS

Mathimatical Operators

Operator Description Example
+ Addition - Adds values on either side of the operator a + b
- Subtraction - Subtracts right hand value from left hand value and returns subtracted value. a - b
* Multiplication - Multiplies values a * b
/ Division - Divides left hand value from right hand value. a / b
% Modulus - Divides left hand value by right hand value and returns the remainder as a value. a % b
** Exponent - Raises the power of the value a**b

Order of Math Operations

The order of operations is:
Parathesis
Exponents
Multiplication
Division
Subtraction

 (a <=> b) is -1 because the return value is false

ex d: Integer === "fortytwo" - False
ex e: /ell/ === 'Hello' - True

Ruby Range Operators

Operator Description Example
.. Creates an inclusive range from a starting point to an ending point 1..10 creates an inclusive range 1 to 10
... Creates an exclusive range from a starting point to an ending point 1..10 creates a range 1 to 9

Conditional Statements

if, elsif, else Statement

number = 50
if number > 50
puts "Your number is too big"
elsif number < 50
puts "Your number is too small"
else
puts "You picked the correct number!"
end

In this example the program goes through each condition and checks for a true statement. The program will stop once a condition is met, otherwise it will continue to loop.

Write your own Methods

def favorite_food(fav_one, fav_two)
sentence = "Your favorite foods are #{fav_one} and #{fav_two}"
puts sentence
end
favorite_food("apples", "oranges")

In the example above favorite_food is a method that accepts two arguments (two favorite foods in this example). The method prints a statement. We call the method and pass arguments after we define our method. "Sentence" is our local variable in this example, it is holding our string that tells us our favorite foods.

pre-defined methods

We don't have to always create our own methods. Ruby has many pre-defined methods that do a lot of work for us.

Resources
Here are some resources to find pre-defined methods: http://www.ruby-doc.org/

Classes

Classes in Ruby are objects. When a class in Ruby is created, an initialized method is defined as well, and we create instances of the class there. Classes allows us to organize our code, so we can use instances of classes in different methods within our class.

For example: If we have a class called Menu, we can assign the menu instance variables. Instance variables are assigned by using @ followed by the name of the variable. Instance variables are attached to the instance of the class. So in our example below: Appetizer, Entree, Dessert, and Drinks are all instances of the class Menu. Our initialize method accepts four arguments, so whenever our program creates a menu, each instance of menu will have an appetizer, entree, dessert, or drink. We can create an instance of our class outside of our class by writing .new after our class name.

class Menu
def initialize(appetizer, entree, dessert, drinks)
@appetizer = appetizer
@entree = entree
@dessert = dessert
@drinks = drinks
end
end
miriams_menu = Menu.new("bread", "chicken", "cake", "tea")
someones_menu = Menu.new("salad", "steak", "tiramisu", "wine")

Reserved Words

Built in words in Ruby cannot be used for naming variables.

Accessors

An object's instance variables are its attributes, the things that distinguish it from other objects of the same class. It is important to be able to write and read these attributes; doing so requires methods called attribute accessors. attr_accessor :var
attr_reader :var
attr_writer :var
