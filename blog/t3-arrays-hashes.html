Arrays & Hashes

October 25, 2014

Hash

A hash is made up of keys and values.
(example: I created a coffee shop menu as a hash. In the example below the drinks are keys, and the prices are values.)

coffee_menu = {
:latte => 5.25,
:pumpkin latte => 5.50,
:coffee => 2.25,
:cappuccino => 3.25,
:americano => 2.50,
:hot chocolate => 3.50
}

Array

An array is a list of objects. These objects can be a string, fixum, or integer.

Example of an array with integers.

[1,2, 3, 4, 5]

You can call an object in an array by indices

[1,2, 3, 4, 5][0]

In this example [0] calls number 1 in the array. You can use strings or integers in an array, but you can only use numbers to get items out of an array

Difference Between Hashes and Arrays

When choosing arrays, you can only use numbers to get items out of it, and with Hashes you can use anything including numbers as an index.
