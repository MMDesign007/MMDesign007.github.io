Ruby on Rails

November 30, 2014

Ruby is an object-oriented programming language developed in 1993 by Yikihiro “Matz” Matsumoto. Ruby is one of the most popular programming languages today, ranking in the top ten.

David Heinemeier Hansson created Ruby on Rails, and first released it in July 2004. It is an open source web application framework written in Ruby. Ruby on Rails, also known as Rails, is an impressive software library that extends the Ruby programming language. Rails will do a lot of the heavy lifting for programmers by reducing the amount of code developers need to write. Rails has strict naming conventions, forcing the programmer to avoid having to repeat him/herself.

Ruby on Rails has it’s own server called WEBrick, apache is also a popular server to use. WEBrick was primarily written by Masayoshi Takahashi and Yuuzou Gotou. Ruby gems is a package manager. Gems provide pieces of code to plug into a Ruby site. You can install rails through ruby gems. A gem has two parts: a code and gemspec file.
