Using Cycle Enumerable in an array

The method .cycle iterates over each item in an array. Once the last item has been passed through the block, the process starts over again. Cycle goes through the array continuously unless it's given a break statement. You can also specify the amount of times you want to cycle through an array.

#This will print every item in the array, and then repeat the process an infinite amount of times.
rainbow = ["blue", "green", "red", "yellow", "orange"]
rainbow.cycle { |x| puts x }
#This will print every item in the array five times.
rainbow = ["blue", "green", "red", "yellow", "orange"]
rainbow.cycle(5) { |x| puts x }
#You can break a statement like this:
rainbow = ["blue", "green", "red", "yellow", "orange"]
rainbow.cycle do |color|
puts color
break if color == "yellow"
end

Using Cycle Enumerable in a hash

# This will print the color and order an infinite amount of times.
colors = {:blue => 1, :green => 2, :red => 3, :yellow => 4, :orange => 5}
colors.cycle {|color| print color}
# Similar to an array, specify the amount of times you want to cycle through the hash, like this. This will print the hash four times.:
colors = {:blue => 1, :green => 2, :red => 3, :yellow => 4, :orange => 5}
colors.cycle(4) {|color| print color}
# Break the cycle like this:
colors.cycle do |color, order|
puts "#{order} : #{color}"
break if color == :red
end
